The following game is an isometric quiz game built with Java

The aim of the game is to move to a computer and answer 5 questions (chosen randomly) correctly in a row.

When you have done this 3 times, you can move to the printer and win.

Compared to the base game, it contains the following extra features
1. Custom tiles and objects
2. Custom sound effects for failure and victory
3. Quiz mechanic - Move to computers and answer questions about programming
